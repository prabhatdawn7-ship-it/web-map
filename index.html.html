📊 Spatial Analysis

Why this topic?
This project analyzes food accessibility around the university.
Objective
To map food stalls, cafes, and fruit shops using GIS.
Methodology
Data collected using latitude & longitude and visualized in Web GIS.
Spatial Pattern
Food vendors cluster near high student movement areas.
Sustainability
Supports local vendors and ensures affordable food supply.
